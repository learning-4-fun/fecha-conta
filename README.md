### How do I get set up? ###

* Dependencies

The environment needs to have the following resources:
NodeJS, ruby 2.1.3, rails 4.1.6 and [Koala](https://github.com/arsduo/koala/wiki/Graph-API)

* Tools

bower: 
```
sudo npm install bower -g
```

* Deployment instructions

```

bundle install
```

```
rake bower:install
```

```
rails s

```
